# 🗡️ CTF Katana

**CTF Katana** is a professional, beautifully crafted Swiss Army Knife for Capture The Flag (CTF) competitions. Built with Python and `rich`, it provides an elegant interactive CLI interface for cryptography, steganography, hashing, and network analysis tasks.

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 🚀 Features

- **Cryptography Module**
  - Encode & Decode constraints (Base64, Hex, URL, Binary)
  - ROT13 Cipher Text translation

- **Hashing Module**
  - Instant Checksums (MD5, SHA-1, SHA-256)
  - Hash Length Indentifier & format guesser

- **Network & Web Module**
  - Multi-threaded Fast Port Scanner (Customize Start/End Ports)
  - Automatic HTTP Header Grabbing and Analysis

- **Steganography & Forensics Module**
  - File Magic Bytes checker (e.g. identify true file format regardless of extension)
  - Extract Strings from Binary Files

- **Web Exploitation & Payloads Module**
  - Analyze and Decode JWT (JSON Web Tokens)
  - Generate Reverse Shells (Bash, Netcat, Python, PHP, PowerShell) for instant copy-pasting

- **Binary Exploitation (Pwn) Module**
  - Generate Cyclic Patterns for Buffer Overflows (like `msf-pattern_create`)
  - Find Exact Offsets using Cyclic Patterns

## 🛠️ Installation

1. Make sure you have Python 3 installed.
2. Clone this repository or download the folder.
3. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## 🎮 Usage

Simply run the Katana script to open the interactive, colorful menu:

```bash
python katana.py
```

Follow the on-screen prompts to easily navigate through tools and conquer your CTF challenges without needing to memorize syntax!
